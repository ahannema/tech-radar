What is My Tech Radar?

My personal Tech Radar is a list of technologies, complemented by skills level, called ring assignment. I use three rings with the following semantics:

  • EXPERT — Technologies where I have expert skills. Technologies that I have used in productive environments and in several projects.
  • COMPETENT — Technologies where I have competent skills. Technologies that I have used in the PoC solutions/development environments and in single projects.
  • BEGINNER — Technologies where I have beginner skills. Technologies that I have used in development environments and mainly in private projects.
